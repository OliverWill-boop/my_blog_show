# 解析

> 解析基本来源于豆包，豆包对于图片内容的识别有些问题，所以说的有些元件和电路图的对不上，但是整体逻辑的讲解还有元件的介绍是没有什么问题的 ，望周知

## **第一步：核心元件认识**

先搞懂每个元件是干嘛的，后面才好理解整体逻辑：

- **GB（R5/R6/R11/R12）光敏电阻**：光照越强，电阻越小；光照越弱，电阻越大。循迹时，**黑线反光弱（电阻大），白纸反光强（电阻小）**，靠它检测黑白差异。

- **R1/R2 可调电位器（500K）**：手动拧它，能改变电压大小，相当于给比较器设一个 **“参考门槛”**，用来判断 “现在光照算强还是弱”。

- IC1/IC2（LM393）电压比较器：有两个输入脚（+ 同相、- 反相），一个输出脚：

  - 若 **+ 脚电压 > - 脚电压** → 输出高电平
  - 若 **+ 脚电压 < - 脚电压** → 输出低电平
  - ⚠️ 它是**开集电极输出**，必须接 “上拉电阻” 才能真正输出高电平。

- Q1/Q2（8550）PNP 三极管：当开关用：

  - 基极电压低（≈0V）→ 三极管导通，电流流过电机 → 电机转
  - 基极电压高（≈3V）→ 三极管截止，电机断电 → 电机停

- **M1/M2 直流电机**：小车的左右轮，分别由 Q1/Q2 控制通断。

- **D1-D5 LED 指示灯**：用来显示电路状态：D4/D5 是电源灯，D1/D2/D3 是电机工作灯，亮 = 电机在转。

- **R3/R4（3.3K）上拉电阻**：给 LM393 的输出 “抬电压”，让它能输出高电平（否则输出是悬空的，控制不了三极管）。

- **C1/C2（100μF）电解电容**：给电源 “滤波”，把电压里的杂波滤掉，让电路更稳定，防止误动作。

- **K1 双刀双掷开关**：总电源开关，控制整个电路通断。

- > 豆包写的有的地方有问题，但是整体介绍没啥问题，就是和电路图对应有点问题，但看懂没问题



## **第二步：模块拆解（从左到右）**

#### 模块 1：电源与滤波电路

- **电源**：DC3V 供电，K1 是总开关，打开后 3V 电压进入电路。

- **滤波**：C1、C2 并联在电源两端，像 “小水库” 一样，把电源里的波动、杂波吸收掉，让电压更平稳。

- **电源指示**：D4、D5 是 LED，串联限流电阻后接在电源上，亮 = 电源正常工作。

  

#### 模块 2：传感器 + 阈值分压电路（以 IC1 为例，IC2 完全一样）

这部分是**把 “光照强弱” 变成 “电压高低”**，是循迹的眼睛👀

**以 IC1 为例：**

- 反相端（-，上面的输入脚）

  ：接光敏电阻 R5 + 固定电阻 R7（1K）的分压点。

  - 公式：`V- = 3V × R7 ÷ (R5 + R7)`
  - 光照强（白纸）→ R5 小 → V- 变大
  - 光照弱（黑线）→ R5 大 → V- 变小

  

- 同相端（+，下面的输入脚）

  ：接可调电位器 R1 + 固定电阻 R8（1K）的分压点。

  - 公式：`V+ = 3V × R8 ÷ (R1 + R8)`
  - 拧 R1 可以改变 V + 的大小，这就是**判断黑白的 “门槛电压”**。

  

**核心逻辑：**

通过调节 R1，把 V + 设成一个固定值（比如 1.5V），让它刚好能区分 “白纸电压” 和 “黑线电压”：

- 白纸（光照强）→ V- > V+

- 黑线（光照弱）→ V- < V+

  

#### 模块 3：比较器输出 + 电机驱动电路（以 IC1→Q1→M1 为例）

这部分是**把 “电压判断结果” 变成 “电机转 / 停”**，是循迹的手脚🦶

 **信号流向：**

1. LM393 输出

   ：

   - 若 V- > V+（白纸）→ 输出**低电平（≈0V）**
   - 若 V- < V+（黑线）→ 输出**高电平（≈3V）**（靠 R3 上拉实现）

   

2. 三极管 Q1 控制

   ：

   - 输出低电平（0V）→ Q1 基极电压低 → PNP 三极管**导通** → 电流从 3V→Q1→电机 M1→地 → 电机 M1 转动，D1 亮（指示）
   - 输出高电平（3V）→ Q1 基极电压高 → PNP 三极管**截止** → 电机 M1 断电，D1 灭

   

## **第三步：整体循迹逻辑（小车怎么沿着黑线走？）**

现在把所有模块串起来，看小车在不同位置时的反应：

### 1. 小车在白纸上（正常前进）

- 左右光敏都在白纸上 → 光照强 → R5/R12 电阻小 → V - 左 > V + 左，V - 右 > V + 右
- IC1/IC2 都输出低电平 → Q1/Q2 都导通 → 左右电机 M1/M2 都转 → 小车**笔直前进**

### 2. 小车向左偏（左光敏压到黑线）

- 左光敏：压到黑线 → 光照弱 → R5 电阻大 → V - 左 < V + 左 → IC1 输出高电平 → Q1 截止 → 左电机 M1 停
- 右光敏：还在白纸上 → 光照强 → R12 电阻小 → V - 右 > V + 右 → IC2 输出低电平 → Q2 导通 → 右电机 M2 转
- 结果：**左轮停，右轮转** → 小车自动**向右转**，慢慢回到黑线上

### 3. 小车向右偏（右光敏压到黑线）

- 左光敏：还在白纸上 → 光照强 → R5 电阻小 → V - 左 > V + 左 → IC1 输出低电平 → Q1 导通 → 左电机 M1 转
- 右光敏：压到黑线 → 光照弱 → R12 电阻大 → V - 右 < V + 右 → IC2 输出高电平 → Q2 截止 → 右电机 M2 停
- 结果：**右轮停，左轮转** → 小车自动**向左转**，慢慢回到黑线上

### 4. 小车完全压在黑线上（停止）

- 左右光敏都压到黑线 → 光照弱 → R5/R12 电阻大 → V - 左 < V + 左，V - 右 < V + 右
- IC1/IC2 都输出高电平 → Q1/Q2 都截止 → 左右电机都停 → 小车**停止**



## **第四步：调试小技巧（新手必看）**

**调节阈值**：先让光敏对着白纸，拧 R1/R2 直到对应电机刚好转动；再对着黑线，电机刚好停止。反复微调，让黑白区分最明显。

**看 LED 判断问题**：

- D4/D5 不亮 → 电源没接通（查 K1 和 3V 供电）
- 电机不转但 LED 亮 → 三极管或电机接线问题
- 电机一直转 → 阈值没调好，或光敏被强光干扰

**光敏位置**：光敏要贴在小车底部，尽量靠近地面，避免环境光干扰，最好加个遮光罩。



## **✨ 一句话总结核心原理**

这个电路靠 **“光敏电阻检测黑白光照→转化为电压差异→LM393 比较判断→三极管控制电机停转→小车自动转向修正”**，纯硬件实现循迹，没有任何单片机，非常直观地体现了 “自动控制” 的基本思想。
