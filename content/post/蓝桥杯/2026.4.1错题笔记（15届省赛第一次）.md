# 2026.4.1错题笔记（15届省赛第一次）

## ne555考点

- 除底层之外有下面几个需要注意的地方
- 1.要注意频率测量的时候用的是定时器0，要注意定时器0和定时器1的初始化先后顺序问题，应该先0再1
- 2.对频率进行数码管显示的时候，要注意1s后再开始显示（定义一个标志位就行），否则会导致第一次频率测量出现问题
- 3.一定要使用跳线帽接上P34和SIGNAL这个线，否则会测量不到东西
- 4.由于按键最后一列会用到P34，所以我们这里必须把他给屏蔽了，不然会导致P34的上升下降测量误差，会
  影响评分

```
	
	在定时器1中断中：
	if(++time1s==1000)
	{
		time1s=0;
		Freq = TH0<<8 |TL0;

```

- 频率就是1s测量到的脉冲上升沿的数量，所以上面在中断中写的代码，就已经取出了频率值

```
void freq_task()
{
	Freq_final = Freq+Freq_jiaozhun;
	if(Freq_final>45000)
		freq_error_flag = 1;
	else
		freq_error_flag = 0;
	
	if(Freq_final>Freq_max)
		echo_flag = 1;
	else
		echo_flag = 0;
	
	if(echo_flag&&(freq_error_flag!=1))
	{
		Freq_max = Freq_final;
		strncpy(ucrtc_echo,ucrtc,3);
	}

	
}
```

- 注意\#include "string.h"，要引用这个头文件，才能用strncpy(ucrtc_echo,ucrtc,3);

- 这个函数的 作用是：将 ucrtc 数组（当前 RTC 时间）的前 3 个字节复制到 ucrtc_echo 数组中。这里的 3 对应 RTC 时间的 3 个组成部分（时、分、秒）。

- if(Freq_final>45000) 能够检测到校准后频率为负数的原因与 C语言中的类型转换规则 有关，具体分析如下：

  ### 变量类型分析
  - Freq ： unsigned int （无符号整型，范围 0~65535）
  - Freq_jiaozhun ： int （有符号整型，范围 -32768~32767）
  - Freq_final ： unsigned int （无符号整型，范围 0~65535）
  ### 关键机制：有符号与无符号的运算转换
  当 Freq_jiaozhun （有符号）与 Freq （无符号）相加时，C 语言会自动将 有符号数转换为无符号数 后再运算。

  负数的无符号表示 ：
   当 Freq_jiaozhun 为负数时，它会被转换为一个 很大的无符号值 （例如，-1 会变成 65535，-1000 会变成 64536）。

  ### 为什么 Freq_final>45000 能检测负数
  - 正常情况 ：如果 Freq_jiaozhun 为正数或较小的负数， Freq + Freq_jiaozhun 的结果会在合理范围内（如 0~45000）。
  - 异常情况 ：如果 Freq_jiaozhun 是 较大的负数 （例如 -10000），转换为无符号数后会变成 很大的值 （如 55536），与 Freq 相加后可能超过 45000。
  ### 举例说明
  假设 Freq = 10000 ：

  - 若 Freq_jiaozhun = 500 （正数），则 Freq_final = 10500 （正常）。
  - 若 Freq_jiaozhun = -500 （小负数），则 Freq_final = 9500 （正常）。
  - 若 Freq_jiaozhun = -20000 （大负数），转换为无符号数后为 65536 - 20000 = 45536 ，则 Freq_final = 10000 + 45536 = 55536 （超过 45000，触发错误标志）。
  ### 结论
  if(Freq_final>45000) 不仅能检测实际频率过高的情况，还能通过 无符号数溢出 机制检测到由于 Freq_jiaozhun 为较大负数导致的校准后频率为负的情况。这是一种巧妙利用 C 语言类型转换特性的错误检测方法。

- 处理频率就在频率函数上运算，显示就在seg上显示，题目没说显示必须在某个条件下才显示，那就不要动数码管模块！！！！



## 默认处理按键有效情景

```
S8、S9 分别定义为 “加”和“减”按键，在参数界面的两个子界面
下有效。
① 超限参数界面下，按下 S8 按键，超限参数增加 1000Hz,按下 S9 按
键，超限参数减小 1000Hz。
② 校准值参数界面下，按下 S8 按键，校准值参数增加 100Hz, 按下
S9 按键，校准值参数减小 100Hz。
```

- 像这种，就需要默认按键8和9是在参数界面（母界面）才有效
